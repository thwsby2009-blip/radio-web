# 網路收音機

一個簡單的網路收音機 Web App，收錄台語、華語、英文電台。

**線上版：** https://thwsby2009-blip.github.io/radio-web

## 收聽頻道

| 頻道 | 類型 |
|------|------|
| 臺北電台 | 台語/國語 |
| Taiwan Lounge | Lounge |
| Ai Radio 台灣 | 華語AI |
| AsiaFM 亞太 | 華語 |
| Jazz Radio | Jazz |
| 瑞士古典 | 古典 |
| SomaFM Drone Zone | 氛圍 |
| FIP Radio | 電子 |
| KEXP 90.3 | 另類 |
| SomaFM Groove Salad | 環境音 |

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，或部署到 GitHub Pages。
